# CharanEduApp – Educational Math & Logic Game

## 🚀 Overview
CharanEduApp is an interactive Android application designed to improve mathematical and logical thinking through engaging puzzle-based gameplay. The app is built for children and early learners, providing a fun and educational experience.

---

## 🎯 Features
- 👤 User name input before starting the game
- 🧩 Image-based math and logic puzzles
- 🔀 Random puzzle order every time
- ✅ Instant feedback (correct / wrong answers)
- 🔊 Sound effects for interaction
- 📊 Score tracking system
- ⏱️ Timer to track gameplay duration
- 🚪 Exit anytime with progress retained
- 💾 Local data storage using Room Database

---

## 🏗️ Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- Navigation Component
